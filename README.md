# Jogo Da Forca

![](https://imgur.com/GZEIWst.gif)

## Descrição 
Jogo da forca desenvolvido em C#, onde o jogador tem que acertar a palavra sorteada, sem ultrapassar o limite de erros.

## Funcionalidades
- Categoria de palavras
- Palavras aleatórias
- Contagem de erros
- Ver letras já chutadas
